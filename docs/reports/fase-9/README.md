# Fase 9 — Backend Administrativo Supabase

## Objetivo

Criar a base administrativa do Xandeflix 2.0 usando Supabase, migrations versionadas e políticas RLS.

## Escopo inicial

- Perfis administrativos
- Clientes
- Dispositivos
- Fontes IPTV
- Cache de canais
- Auditoria
- Regras de acesso com RLS

## Restrições

- Não alterar Player Universal nesta etapa.
- Não alterar navegação Fire Stick nesta etapa.
- Não expor service_role no frontend.
- Toda tabela administrativa deve ter RLS.
- Mudanças devem ser pequenas, revisáveis e reversíveis.
